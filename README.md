# Splitwise Clone

A modern, elegant, and minimal web application for splitting expenses and managing group debts, built with React, Node.js, Express, and MySQL.

## Setup Instructions

### Prerequisites
- Node.js (v18+)
- MySQL Database

### Installation
1. Clone the repository.
2. For backend:
   ```bash
   cd backend
   npm install
   ```
3. For frontend:
   ```bash
   cd frontend
   npm install
   ```
